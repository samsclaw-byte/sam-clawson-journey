üåÖ Morning Brief

Daily 6:00 AM automated briefing with recovery data, tasks, and recommendations

What It Does

The Morning Brief is your daily automated briefing that runs at 6:00 AM, pulling data from multiple sources to give you a complete picture of your day ahead.

Automated Daily 6:00 AM

Workflow Steps

  1. Fetch TAT Tasks
    Pulls Category 1 (Today) and overdue tasks from Airtable Productivity base
  2. Get WHOOP Recovery
    Retrieves recovery score, sleep performance, HRV, and resting heart rate
  3. Analyze Yesterday's Health
    Shows calories consumed, habits completed, and workouts logged
  4. Generate Workout Recommendation
    Based on recovery zone (Green/Yellow/Red), suggests appropriate workout type
  5. Save Brief to File
    Stores markdown summary for historical reference

Recovery Zones

Zone Score Recommendation
üü¢ Green 67%+ Full intensity workout (30-45 min)
üü° Yellow 50-66% Active recovery (swim, easy bike, yoga)
üî¥ Red <50% Complete rest or mobility only

Script Location & Usage

üíì WHOOP Integration

Automatic recovery and sleep data via webhooks

How It Works

  1. Wear WHOOP
    Device continuously monitors heart rate, HRV, sleep, and strain
  2. Data Sync
    WHOOP sends webhooks to our server when data updates
  3. Save to Airtable
    Recovery scores, sleep performance, and HRV stored in Health base
  4. Morning Brief
    Used to generate workout recommendations

Data Collected

Metric Description
Recovery Score 0-100% based on HRV, RHR, sleep
Sleep Performance Actual sleep vs. sleep needed
HRV Heart rate variability (ms)
Resting HR Resting heart rate (bpm)
Sleep Duration Total time in bed (hours)
